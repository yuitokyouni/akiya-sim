# mig 実データ差替え — 検証記録

## データ

- 出所: [人口の動き（令和6年中）第1表](https://www.toukei.metro.tokyo.lg.jp/jugoki/2024/ju24q10000.htm)
- CSV: `data/raw/ju24qv0100.csv`（取得日: 2026-07-10）
- 集計: `scripts/build_zones_mig.py` → `data/zones_mig.csv`
- 定義: `mig = (他県移動増減 + 都内間移動増減) / 期初期末平均人口`
