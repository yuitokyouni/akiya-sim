# 都民主語プロダクト（主人公の交代）

## 一言

プロダクトを作り直さない。**問いを変える。**  
「政策は効くか？」→「うちの街は10年後どうなってる？」

## 4枚

1. **いま** — 実データ（空き家・高齢化・取引）。予測禁止。  
2. **このまま10年** — シミュレーションは**レンジ**。断定禁止・前提明示。  
3. **なぜ** — 相続分岐・放置・更新の仕組み。  
4. **変えられること** — 区の制度＋政策つまみ。現行シミュはここに格納。

## 禁止

- 地域間ランキング  
- 「あなたの街は衰退します」型の断定  
- 診断だけで終わる画面（必ず処方＝4枚目）

## 工数

モデル 3 : データ 3 : フロント 4（フロントの大半はコピーライティング）

## 依存

実名地名に数字を出す前に、観測値キャリブが必須。  
現状の6地域フィットだけでは町丁目実名表示は出さない。

## 「いま」観測パック（実装済み）

- 生成: `npm run build:ima` → `data/muni_obs_ima.{csv,json,gen.js}`
- ゲート: `npm run harness:muni-ima`
- 説明: `docs/validation_muni_obs_ima.md`
- 検索単位: 市区町村／主要駅まで（町丁目の空き家率は未収録）
